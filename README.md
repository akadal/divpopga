# divpopga
Diverse Population Genetic Algorithm for Real-valued Optimization

# Installation

```julia
] add https://github.com/jbytecode/divpopga
```
or 

```julia
julia> using Pkg
julia> julia> Pkg.add(PackageSpec(url = "https://github.com/jbytecode/divpopga"))
```

# Update

```julia
] update
```

or

```julia
julia> using Pkg
julia> Pkg.update()
```
